# Student Performance Analysis

## Overview

This project analyzes the Student Performance dataset using Python, Pandas, and Matplotlib. It demonstrates the complete data analysis workflow, including data loading, cleaning, exploration, analysis, and visualization using Jupyter Notebook.

## Features

* Load and inspect the Student Performance dataset
* Check for missing values and duplicate records
* Analyze students' final grades (G3)
* Find the number of students scoring above 15
* Study the correlation between study time and final grades
* Compare average performance by gender
* Visualize data using histograms, scatter plots, and bar charts

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Dataset

The project uses the student-mat.csv dataset from the Student Performance Dataset

## How to Run

1. Clone this repository.
2. Install the required libraries:

   ```
   pip install pandas matplotlib
   ```
3. Open `Student_Performance_Analysis.ipynb` in Jupyter Notebook or JupyterLab
4. Run all the cells to reproduce the analysis and visualizations
